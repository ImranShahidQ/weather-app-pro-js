# 🌤 Weather App (JavaScript)

A modern weather application built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

---

## ✨ Features

* Search weather by city name
* Real-time weather data using API
* Temperature, humidity, and wind speed
* Weather icons and description
* Error handling for invalid cities
* Clean and modern glass UI

---

## 🔐 Security Note

The API key is stored in a separate `config.js` file and is excluded from version control using `.gitignore`.

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (Fetch API, Async/Await)
* OpenWeatherMap API

---

## 📂 Project Structure

```bash
weather-app/
 ├── index.html
 ├── style.css
 ├── script.js
 ├── config.js   (ignored in Git)
 └── images/
```

---

## ⚙️ Setup Instructions

1. Clone the repository
2. Create a `config.js` file
3. Add your API key:

```javascript
const API_KEY = "your_api_key_here";
```

4. Run the project in browser
